Bioinformatics & Data Science

Tools and resources for biological data analysis that form the foundation of my interdisciplinary approach.

Genomics & Transcriptomics

  • Bioconductor - Tools for analysis of high-throughput genomic data
  • BioPython - Tools for biological computation in Python
  • Trinity - RNA-Seq de novo assembly
  • KEGG - Database for understanding high-level functions of biological systems
  • GenBank - Genetic sequence database

Protein Analysis

  • ESM-2 - State-of-the-art protein language model
  • ProtT5 - Transformer models for protein sequence analysis
  • AlphaFold DB - Protein structure database
  • BLAST - Sequence similarity search tool
  • InterPro - Protein classification database

Data Visualization

  • Plotly - Interactive visualization library
  • Matplotlib - Comprehensive visualization in Python
  • ggplot2 - Grammar of graphics for R
  • D3.js - JavaScript library for data-driven documents
  • Cytoscape - Network visualization platform

Statistical Analysis

  • scikit-learn - Machine learning in Python
  • tidyverse - Collection of R packages for data science
  • DESeq2 - Differential gene expression analysis
  • pandas - Data analysis and manipulation tool
  • NumPy - Fundamental package for scientific computing
